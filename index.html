A autenticação em um primeiro momento foi feita utilizando o sistema Samba 4, porém depois de algumas dificuldades de tentativas e erros, não resultando em um funcionamento adequado do sitema foi substituida por outro serviço. Depois de um tempo pesquisando sobre alguns serviços de autenticação surgiu como alternativa o Protocolo LDAP(Lightweight Directory Access Protocol) que não é em si um serviço de autenticação, mas juntamente com o padrão X.500(ramificação através de camadas) e com o TCP/IP, permitem que acessem um cliente, consumam dados a partir de uma busca e aí sim é possível a autenticação de usuários(SLAPD = LDAP + X.500). Para fazer a autenticação o LDAP busca as informações do servidor SLAPD que compartilha as mesmas informações do Banco de Dados, depois dessa busca utiliza-se bibliotecas para fazer a autenticação(LIBPAM-LDAP e a LIBNSS-LDAP).
